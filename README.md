# APP DE PELICULAS (BASICO)

Proyecto educacional para gente que quiere aprender como se hace un APP de peliculas en kotlin.

El proyecto fue realizado con fines educativos
### Tecnologias usadas

- Glide
- FireStore Database
- Live data
- View Model
- View Binding
